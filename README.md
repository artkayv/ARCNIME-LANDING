A free, ad-supported anime streaming platform designed with a clean and minimal UI. This repository contains the frontend code for the **Official Mirror Server Selector**, allowing users to check real-time server availability and ping times before watching.

## ✨ Features

* **Real-time Status Checking:** Automatically pings each mirror server every 5 seconds to display live latency (`ms`) and online/offline status.
* **Minimalist UI:** A sleek, responsive Black & Yellow design built for clarity and ease of use.
* **Zero Dependencies:** Built entirely with pure HTML, CSS, and Vanilla JavaScript. No bulky frameworks or libraries.
* **Dynamic Footers:** Automatically grabs and displays the current hostname.

## 🚀 Quick Start (Local Development)

Since this project uses pure HTML/JS/CSS, there is no build step required.

1. Clone or download this repository.
2. Open the `arcnime-selector` folder.
3. Double-click on `index.html` to open it in your default web browser.

*Note: The real-time ping feature uses `no-cors` fetch requests. While it will work locally, testing it in a live environment is recommended for accurate latency readings.*

## 🛠️ Tech Stack
